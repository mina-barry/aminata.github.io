À partir d’une **maquette fournie**, j’ai réalisé un site web composé d’une
**longue page d’accueil**.  
J’ai travaillé le **responsive design** afin que le site s’adapte aux différents
écrans (ordinateur, tablette, mobile).

Sur ce même projet, j’ai également utilisé **Tailwind CSS** pour réaliser
une partie du site et gérer le responsive avec ses classes utilitaires.

**Technologies utilisées :**
- HTML
- CSS
- Tailwind CSS

**Compétences développées :**
- Intégration d’une maquette
- Responsive design
- Organisation du code CSS
